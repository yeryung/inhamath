## git 브런치 생성
<br>

```
git branch [브런치명]
git branch yeryung
```

<br><br>

## 생성된 브런치로 이동
* git checkout yeryung, checkout에서 변경되었다.
<br>

```
git switch [브런치명]
git switch yeryung
```
